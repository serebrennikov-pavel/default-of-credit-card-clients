# Default of Credit Card Clients documentation!

## Description

Разработка и внедрение сквозного (end-to-end) решения для оценки кредитных рисков. Проект включает в себя создание модели классификации для прогнозирования дефолта клиентов, её упаковку в микросервис с использованием Docker и реализацию инфраструктуры для проведения A/B-тестирования версий модели в production-среде.

## Commands

The Makefile contains the central entry points for common tasks related to this project.
